## Pre-manifesto

### 25 January → 8 February 2026 — Synthesis

Convergence of all deconstructed principles into a single paradigm. Key concepts crystallized: Separation of Volatility (replacing Separation of Concerns), Constraint Corridor, specification bottleneck as formal thesis. "EVOLUTIONARY ZERO" formally named — philosophy separated from product. System 0/1/2/3, DHS Register Thesis, ZLC protocol, and Bottleneck Shift thesis formalized in working notes. This period produced the intellectual foundation that became v1.0.

### 24–25 January 2026 — Deconstruction

Five parallel analyses of the same seven legacy software principles, each written for a different audience (backend developers, system architects, startup founders, non-linear thinkers). Core insight discovered across all five: cost of generation collapsed → bottleneck = selection, not creation → human rises up the abstraction stack. Key concepts first articulated: intent as asset, code as disposable, human as selector, constraint-guided exploration, Probabilistic Intent Engineering, Assumption Ledger, Separation of Volatility.

### 2–24 January 2026 — Origin

Seed of the entire concept: a psychologically grounded architecture for AI as cognitive partner (not tool), combining six scientific methodologies. First appearance of the future System 0 role — human must rise to meta-level to steer dialogue. Early working conversations explored first principles and cognition, non-linear thinking as a different cognitive OS, naming experiments, and state-based mode switching. On 23 January, the core thesis emerged: "intent as asset, code as disposable" — specification over execution.
